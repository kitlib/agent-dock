# Skill 管理功能待办清单

## 说明

本文档基于当前代码状态整理，不再沿用旧的“Marketplace 尚未接入”判断。

当前对 Skill 的判断应更新为：

- 本地 Skill 管理第一阶段已完成
- `skills.sh` Marketplace 真实链路已完成并接入 Home
- 仍缺少关系视图和更完整的批量管理能力

## 已完成

### A. 全局本地 Skill 资源池（第一阶段）

当前已实现：

- 左侧 Agent 列表顶部提供“全部”入口
- 选中“全部”后，聚合所有已管理且未隐藏 Agent 的本地 Skill
- 中栏复用现有 Skill 列表视图
- 右栏支持“全部 Skills”空选中态说明
- 全局视图下，Skill 列表显示归属 Agent 标签

当前边界：

- 仅 Skill 接入“全部”聚合
- MCP / Subagent 暂未接入

结论：

- 已完成

### B. 本地 Skill 复制闭环（基础版）

当前已实现：

- 支持批量复制本地 Skill 到其他 Agent
- 支持单个 Skill 从列表“更多”菜单发起复制
- 复制目标使用平铺 Agent 卡片
- 支持多选目标 Agent
- 支持冲突预览
- 支持 `overwrite / skip`
- 复制完成后刷新 Skill 列表与 Agent 列表

当前边界：

- 复制完成后的结果反馈仍偏轻量
- 尚未提供复制后自动定位到目标 Agent 或目标 Skill

结论：

- 已完成基础闭环

### C. 本地 Skill 操作体验优化

当前已实现：

- 列表“更多”菜单顺序统一为：打开、编辑、复制、停用 / 启用、删除
- 右栏详情操作顺序统一为：访问 / 安装、更新、打开、编辑、停用 / 启用、删除
- 详情侧删除操作具备确认弹窗
- 复制弹窗保留稳定高度，减少预览抖动

当前边界：

- 列表项中的删除入口仍未加确认

结论：

- 已完成大部分交互顺序优化

### D. Skill Marketplace 真实接入

当前已实现：

- `skills.sh` 排行榜
- `skills.sh` 搜索
- Skill Marketplace 详情
- Marketplace 安装预览
- 安装到指定 Agent
- 覆盖安装
- 安装记录持久化
- 详情缓存
- 本地 Marketplace Skill 更新检查
- 本地 Skill 详情中的更新入口
- 安装完成后刷新本地 Skill 池

当前边界：

- 仅覆盖 Skill
- `mcp / subagent` 仍是 mock

结论：

- 已完成 Skill Marketplace 接入

## P1

### 2. “被哪些 Agent 使用”关系视图

目标：

- 从“Skill 属于哪个扫描来源”扩展到“Skill 被哪些 Agent 使用”的视角

当前缺口：

- 现有 `ownerAgentId` 仅表示扫描归属
- 没有显式的多 Agent 绑定 / 使用关系模型

验收标准：

- 能区分“来源 Agent”和“使用 Agent”
- 至少先提供一套只读关系视图

复杂度：

- 中到高

风险：

- 中

### 3. 批量删除

目标：

- 补齐本地 Skill 管理的基础批量删除能力

当前缺口：

- 已有批量启停
- 已有批量复制
- 缺少批量删除

验收标准：

- 可对选中的多个本地 Skill 执行删除
- 删除前有明确确认
- 删除后列表与详情状态正确刷新

复杂度：

- 低到中

风险：

- 中

### 4. 复制结果反馈增强

目标：

- 让复制能力从“可用”提升到“可管理”

当前缺口：

- 已有冲突预览
- 结果反馈仍主要依赖 toast

验收标准：

- 清晰显示复制成功、跳过、覆盖结果
- 复制后可快速定位到目标 Agent 或目标 Skill

复杂度：

- 低

风险：

- 低
