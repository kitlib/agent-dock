# AgentDock 多 Agent 资源管理功能设计

## 功能定位
AgentDock 的主功能是一个面向多 Agent 的本地资源管理中心，用于统一管理各 Agent 绑定的 Skills、MCP 和 Subagents。Marketplace 是辅助导入能力，用于发现、安装并导入资源到本地资源池，不是主工作台。

## 核心功能

### 1. 多 Agent 管理
支持：
- Agent 列表展示
- 新增 Agent
- 删除 Agent
- 重命名 Agent
- 切换当前 Agent
- 查看每个 Agent 的资源概览

### 2. 本地资源池管理
统一管理本地资源：
- Skills
- MCP
- Subagents

支持：
- 查看资源列表
- 查看资源详情
- 查看启用状态
- 删除资源
- 从本地导入资源
- 查看资源被哪些 Agent 使用

### 3. Agent 资源绑定管理
支持：
- 为 Agent 添加资源
- 从 Agent 移除资源
- 启用 / 禁用绑定资源
- 调整顺序
- 查看绑定状态

### 4. 分组管理
支持：
- 创建分组
- 重命名分组
- 删除分组
- 折叠 / 展开分组
- 分组排序
- 删除分组后将资源移回未分组区

### 5. 拖拽编排
支持：
- 未分组区内排序
- 分组内排序
- 从未分组拖入分组
- 跨组拖拽移动
- 从分组拖回未分组区

### 6. 批量操作
支持：
- 批量启用
- 批量停用
- 批量移动到分组
- 批量移出分组
- 批量解除绑定
- 可选批量删除本地资源

### 7. Marketplace 辅助导入
支持：
- 浏览市场资源
- 搜索资源
- 查看资源详情
- 从市场安装资源到本地资源池
- 安装后立即绑定到某个 Agent
- 或仅加入本地资源池，稍后再绑定

### 8. 市场源配置
支持：
- 新增 source
- 编辑 source
- 删除 source
- 启用 / 禁用 source
- 健康检查
- 默认内置 `skillhub`

## 页面结构

### Agents
主工作台，用于：
- 切换 Agent
- 查看当前 Agent 的资源编排
- 执行启停、分组、拖拽和批量操作

### Resources
本地资源池页面，用于：
- 查看所有本地 Skills、MCP、Subagents
- 查看资源详情
- 删除资源
- 导入资源
- 查看资源使用关系

### Marketplace
辅助导入页面，用于：
- 浏览市场资源
- 搜索资源
- 查看详情
- 安装到本地资源池
- 安装后绑定到 Agent 或仅加入资源池

## 首期范围
首期包含：
- 多 Agent 管理
- 本地资源池
- Agent 资源绑定
- 分组管理
- 拖拽排序
- 批量启停与批量移动
- skillhub 接入
- Marketplace 导入资源到本地池
- 安装后绑定到 Agent
- source 配置管理

首期不包含：
- 自动切换 Agent 资源组合
- 多 Agent 联动编排
- 自动依赖安装
- MCP 在线安装
- Subagents 在线安装
- 多 provider 聚合搜索
- 云端同步
- 评分、评论、推荐系统

## 使用原则
- 日常主操作在 Agents 页面完成
- Marketplace 只负责资源发现和导入
- 安装后的资源统一进入本地资源池
- 是否被某个 Agent 使用，由 Agent 管理页决定
- 删除绑定与删除资源必须明确区分
- 高风险删除操作需要确认
- MCP 敏感配置默认不明文展示
